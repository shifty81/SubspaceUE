KEY FEATURES:
════════════

1. MATERIAL TABS
   - Iron (Grey), Titanium (Silver-Blue), Naonite (Green)
   - Trinium (Blue), Xanion (Gold), Ogonite (Orange-Red)
   - Avorion (Purple)
   - Color-coded text matching in-game materials

2. BLOCK SELECTION
   - Visual grid layout (8 columns)
   - 15 block types total
   - Click to select
   - Hover for tooltip
   - Selected block highlighted

3. RESOURCE TRACKING
   - Real-time inventory display
   - Color-coded amounts
   - Red for zero quantities
   - Credits prominently displayed

4. SHIP STATISTICS
   - Live updates as blocks added/removed
   - Mass, volume, power calculations
   - Propulsion metrics
   - Defense capabilities

5. BUILD CONTROLS
   - Grid snap toggle
   - Variable block sizes
   - Precise positioning
   - Mode switching (Place/Remove)

WINDOW POSITIONING:
═══════════════════

Resource Panel:  Top-left corner    (10, 10)
Main Window:     Screen center      (CenterX, CenterY)
Stats Panel:     Right side         (ScreenWidth - 310, 10)

All windows use semi-transparent backgrounds (alpha: 0.9-0.95)
No title bars for immersive feel
No resize handles for consistent layout

INTERACTION FLOW:
════════════════

1. Open builder (B key)
2. Select material (click tab)
3. Select block type (click in grid)
4. Adjust size if needed
5. Position cursor
6. Place block
7. Check stats panel for ship updates
8. Save blueprint when done

KEYBOARD SHORTCUTS:
══════════════════

B         - Toggle builder UI
1-5       - Quick tool selection (future)
ESC       - Close builder
Mouse     - Navigate and select

TECHNICAL NOTES:
═══════════════

- Built with ImGui
- Multiple coordinated windows
- Material colors from MaterialProperties.cs
- Block types from BlockType enum
- Integrates with BuildSystem.cs
- Compatible with VoxelStructureComponent
- Blueprint save/load system included
